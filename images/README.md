# Carpeta de Imágenes

Esta carpeta debe contener las siguientes imágenes para que el sitio web funcione correctamente:

## Imágenes requeridas:

### Logos
- `logo-kudal.png` - Logo principal (fondo claro)
- `logo-kudal-white.png` - Logo para fondo oscuro (footer)

### Hero Slider (Carrusel principal)
- `hero-1.jpg` - Imagen de fondo slide 1 (recomendado: 1920x600px)
- `hero-2.jpg` - Imagen de fondo slide 2 (recomendado: 1920x600px)
- `hero-3.jpg` - Imagen de fondo slide 3 (recomendado: 1920x600px)

### Sectores (Logística)
- `sector-energia.jpg` - Energías renovables (recomendado: 800x600px)
- `sector-alimentacion.jpg` - Alimentación (recomendado: 800x600px)
- `sector-moda.jpg` - Moda (recomendado: 800x600px)
- `sector-agroalimentacion.jpg` - Agroalimentación (recomendado: 800x600px)
- `sector-industrial.jpg` - Industrial (recomendado: 800x600px)
- `sector-construccion.jpg` - Construcción (recomendado: 800x600px)

## Notas:
- Todas las imágenes deben estar optimizadas para web
- Formato recomendado: JPG para fotos, PNG para logos con transparencia
- Tamaño máximo recomendado: 500KB por imagen para mejor rendimiento

## Imágenes temporales:
Mientras no tengas las imágenes definitivas, puedes usar servicios como:
- https://unsplash.com (fotos de alta calidad gratuitas)
- https://pixabay.com (imágenes libres de derechos)
- https://pexels.com (fotos gratis de stock)

O usar placeholders temporales como:
- https://placeholder.com
- https://picsum.photos
